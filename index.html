Projetos

Easybank landing page

A Página de Destino Easybank é um projeto de desenvolvimento web que combina HTML, CSS e JavaScript para criar uma interface de usuário atraente e funcional para o fictício serviço de banco online chamado "Easybank". Este projeto tem como objetivo fornecer uma experiência de usuário agradável e informativa, destacando os recursos e serviços oferecidos pela instituição financeira fictícia

Insure landing page

A Página de Destino Insure é um impressionante projeto de desenvolvimento web que faz uso das linguagens HTML, CSS e JavaScript para criar uma landing page sofisticada e envolvente para a fictícia empresa de seguros "Insure". O objetivo primordial desta página é cativar e informar visitantes em potencial sobre os serviços de seguro oferecidos, incentivando-os a agir, como solicitar uma cotação de seguro, enquanto transmite a credibilidade e confiabilidade da marca.
